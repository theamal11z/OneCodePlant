üîß Usage Examples

Building a ROS2 Workspace

# Build entire workspace python -m onecode.cli build workspace # Build specific packages python -m onecode.cli build workspace --packages my_robot navigation # Build with debug output python -m onecode.cli build workspace --debug --parallel-jobs 4

Simulation Management

# Start simulation with default world python -m onecode.cli sim start # Start with specific world and robot python -m onecode.cli sim start --world office.world --robot turtlebot3 # Run in headless mode for CI/CD python -m onecode.cli sim start --headless
